# Notifications_app# הוראות שימוש

אלו קובצי המקור לאפליקציית אנדרואיד לקבלת התראות מ-n8n.
פתח פרויקט חדש ב-Android Studio והחלף את הקבצים שנוצרו בקבצים אלו:
- MainActivity.kt הולך ל- java/com/n8n/notifications/
- AndroidManifest.xml הולך לתיקיית main/
- activity_main.xml הולך ל- res/layout/
- build.gradle מחליף את ה-build.gradle של המודול (app)
